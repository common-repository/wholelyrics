Display the selected lyric to your blog with customizable colors and scroll speed.

== Description ==

This plugin will allow you to display your favorite lyrics to your blog.

== Installation ==

1. Unzip the file to your '<blog directory>/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress Admin.
3. Go into Plugins->Whole Lyircs to see the configuration page.
